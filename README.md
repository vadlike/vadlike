## About

I build Windows-first tools, automation workflows, and desktop utilities that solve practical problems and remove repetitive manual work.

My focus is on software that is simple to run, fast to understand, and genuinely useful in day-to-day workflows.

## Focus Areas

- PowerShell automation and workflow scripting
- Python utilities and support tooling
- C# desktop applications for Windows
- System integration, launchers, wrappers, and productivity tools

## Selected Projects

- [VAD Control Suite](https://github.com/vadlike/VAD-Control-Suite): Tray-first Windows control center for monitor blackout, desktop layouts, window tools, quick launcher folders, jiggler modes, and scheduled power actions.
- [NanoKVMPro USB Graphic Driver Wrapper](https://github.com/vadlike/nanokvmpro-usb-graphic-driver-wrapper): Ready-to-use Windows driver wrapper with a simplified install and removal flow for the NanoKVM Pro USB graphic driver.
- [mkcert GUI VADLIKE](https://github.com/vadlike/mkcert-GUI-VADLIKE): GUI wrapper for `mkcert` that makes local certificate generation easier for development workflows on Windows.
- [NVIDIA App Downloader](https://github.com/vadlike/NvidiaApp-downloader): PowerShell-based silent downloader and installer flow for the NVIDIA App with a lightweight folder selection UI.
